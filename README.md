# Text styling app (project - 2)

This Text Styler project is a React + TypeScript application that allows users to input text and apply different font styles dynamically. 


<small><strong> The main features include: </strong></small>
<ul>
 <ol><strong> Live Text Styling: </strong> Users can type text and apply various font styles by clicking buttons. </ol>

# BMI Calculator (project - 4)
This  BMI Calculator is a React Typescript application that allows user to input the weight and height, calculates BMI and tells if a person is obese or underweight or underweight or normal weight.


<small><strong></strong></small>
<ul>
 <ol> <strong>BMI Calculation: </strong> Users can enter their height (in cm) and weight (in kg).The application calculates the Body Mass Index (BMI).</ol>
 <ol> <strong>BMI Status Message:</strong> Based on the BMI value, the app shows:
  <ol>Underweight (BMI < 18.5)</ol>
  <ol>Normal weight (18.5 ≤ BMI < 24.9)</ol>
  <ol>Overweight (25 ≤ BMI < 29.9)</ol>
  <ol>Obese (BMI ≥ 30)</ol>
 </ol>
</ul>

# In the terminal:

<small><strong>create the app first: </strong></small>
```
npm create vite@6.2.1
```
<small><strong>Open the folder that the app is created: </strong></small>
```
cd app name
```
<small><strong>To install the nodes: </strong></small>
```
npm install
```
<small><strong>To download the bootstrap (front-end framework) : </strong></small>
```
npm install bootstrap@5.3.3
```
